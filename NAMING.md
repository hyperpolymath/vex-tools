# VEX Toolkit Naming Policy

## Canonical Product Name

- **ISA** (Irritation Surface Analyser) is the canonical product name.

## Legacy Compatibility Name

- **Vexometer** is the legacy codename for ISA.
- The `vexometer` identifier remains valid and intentionally retained for compatibility in:
- repository paths (`vexometer/`, `vexometer-satellites/`)
- package/module names (`Vexometer.*`)
- CLI command names (`vexometer`)
- protocol IDs (`vexometer-trace-v1`, `vexometer-efficacy-v1`, `vexometer-metrics-v1`)

## Documentation Rule

- In user-facing prose, prefer **ISA**.
- On first mention in a document, write **ISA (formerly Vexometer)**.
- Use **vexometer** only when referring to compatibility identifiers.
