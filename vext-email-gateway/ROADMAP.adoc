= vext-email-gateway Roadmap

== Current Status

Prototype stage. Core architecture and message flow are drafted, but production wiring is not complete.

* [ ] Align cryptography/data model dependencies with current crate APIs
* [ ] Replace prototype identity handling with persistent email-to-identity storage

* [ ] Add compile + test CI gate and ensure green baseline

* [ ] Implement SMTP/IMAP ingestion loop
* [ ] Complete subscription persistence and migration model
* [ ] Add robust error taxonomy and retry/backoff policies

* [ ] Security review completed and documented
* [ ] Rate limiting, abuse controls, and audit logging enabled
* [ ] Operational runbook and deployment guide published
